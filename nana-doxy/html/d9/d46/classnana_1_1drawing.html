Detailed Description

Draw pictures on a widget by specifying a drawing method that will be employed everytime the widget refreshes.

By the end of drawing, the picture may not be displayed immediately. If a picture need to be displayed immediately call nana::gui::API::refresh_window() .

Examples:
a_group_impl.cpp, background-effects.cpp, draw.cpp, and stretch_image.cpp.

Detailed Description

template<typename DrawerTrigger, typename Events, typename Scheme>
class nana::widget_object< category::lite_widget_tag, DrawerTrigger, Events, Scheme >

Base class of all the classes defined as a non-graphics-buffer widget window. The second template parameter DrawerTrigger is always ignored.
